How it works

Three steps. No setup. No software install.

01

Describe it

Type what you need in plain English. Geometry, dimensions, materials. Try flanged cylinder, 50mm tall or L-bracket, 3mm steel.

02

Generate

Designmatix converts your description into Python CAD code, then builds the geometry using OpenCascade — the same engine that powers FreeCAD. Takes 5 to 15 seconds.

03

Download

Get a real STEP file (ISO 10303) plus STL, ready to open in SolidWorks, Fusion 360, FreeCAD, or your slicer. Inspect the generated Python code if you want to tweak it further.

What it makes

From fasteners to fixtures, in seconds.

Fasteners & hardware

Why bother

Built for engineers who already know CAD.

Real STEP files, not pretty pictures.

Output is ISO 10303 STEP — the format every CAD package reads. Open in SolidWorks, Fusion, FreeCAD, NX, Catia. Modify the geometry, add to assemblies, send to manufacturing. It's not a screenshot or a mesh; it's parametric solid geometry.

You see the code that built it.

Every generation produces real CadQuery Python. If the result isn't quite right, inspect the code, learn what went wrong, edit and re-run. Designmatix is a starting point, not a black box.

Bring your own AI key.

Designmatix is free to use — you provide your own Anthropic API key (around $5 of credits handles dozens of generations). Your key never leaves your browser session. We don't store it, log it, or use it ourselves.
